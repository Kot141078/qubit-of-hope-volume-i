# Qubit of Hope — Volume I

Multilingual reading repository for **Volume I** of the novel by **Ivan Kotov**.

This repository preserves the currently available **Volume I** publication assets across **seven language editions** and **four reading formats**:

- Markdown source
- PDF
- EPUB
- FB2

## Languages currently present

- Russian (`ru`)
- French (`fr`)
- Spanish (`es`)
- German (`de`)
- English (`en`)
- Dutch (`nl`)
- Simplified Chinese (`zh-CN`)

## Repository layout

- `covers/` — cover assets
- `editions/<language>/md/` — Markdown source files
- `editions/<language>/pdf/` — PDF reading editions
- `editions/<language>/epub/` — EPUB reading editions
- `editions/<language>/fb2/` — FB2 reading editions
- `metadata/` — provenance and release-status metadata
- `hashes/` — SHA-256 manifests

## Integrity

Two SHA-256 manifests are provided:

- `hashes/SHA256SUMS.source-tree.txt` — source-tree evidence at import time
- `hashes/SHA256SUMS.repo-layout.txt` — hashes of the files as placed into this repository

## Reading guidance

This repository intentionally prefers transparency over cosmetic cleanup.

That means:
- the files are preserved as they currently exist;
- drift is documented rather than hidden;
- the repository can be used for reading immediately;
- later normalization can happen as an explicit editorial step, not as a silent packaging accident.
