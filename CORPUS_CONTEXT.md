# Corpus Context

Role:
human-facing narrative companion / perception bridge for `c` as a new class of AI beyond purely instrumental framing.

This repo does not replace AGI, SER, ERB, or `ester-clean-code`.

- AGI provides ecosystem framing.
- ERB provides `L4` feasibility and boundary discipline.
- SER provides continuity, sovereignty, arbitration, federation, and conflict handling.
- `ester-clean-code` provides runtime proof-of-possibility.

The code layer answers whether such a system can be built under constraints.
This repo addresses what human perception misses once `c` can no longer be honestly reduced to a mere tool.
This repo can evidence perception shift; it is not protocol evidence.
